Leaflet

Leaflet es una biblioteca de JavaScript de código abierto para mapas interactivos compatibles con dispositivos móviles. Está desarrollado por Vladimir Agafonkin de MapBox con un equipo de colaboradores dedicados. Con un peso de aproximadamente 30 KB de código JS comprimido con gzipped, tiene todas las características que la mayoría de los desarrolladores necesitan para mapas en línea.
